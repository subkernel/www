Mori. Mobile web browser for Mac.

The best way to see the mobile websites on Mac.

Mori is a web browser built to view mobile websites and webapps on Mac. It works just like iPhone Safari. Just click safari extension button to open mobile version of the website. Also, Mori supports various modes like popover in menu bar, column browsing, floating window with transparency.

Menubar popover
  • ⌘ + p to pin to menu bar.
  • Drag popover to detach to window.
  • Option + click to open main menu on main icon.

Column, The new way of web browsing.

Column browsing
  • ⌘ + click to open a link in a new column.
  • ⌘ + Shift + click to open a link in the next column.
  • ⌘ + Option + click to open a link in a new window.
  • Drag over a window on another window to attach.
  • Long press titlebar to detach to new window or realign.
Basic gestures and actions
  • Double click titlebar to scroll top.
  • Swipe right to go back.
  • Swipe left to go forward.
  • Pull down to refresh. (Only on the top of webpages)
  • Click address bar to search or open favorites.
